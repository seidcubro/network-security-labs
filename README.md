# Networking and Security Labs

Academic labs and exercises focused on computer networking and security
fundamentals.

## Topics Covered
- Networking fundamentals
- Network architecture
- Basic security principles
- System communication concepts
- Protocols and data transmission

## Tools & Technologies
- Command Line
- Networking utilities
- Linux
- Git

## Structure
Labs are organized by topic and coursework requirements.

## Purpose
This repository demonstrates foundational understanding of networking and
security concepts relevant to modern computing environments.
